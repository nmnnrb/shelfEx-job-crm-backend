
# zippee-ass

A small React app scaffolded with Vite. This README explains how to get the project running locally and describes the important scripts and project layout.

Quick start
-----------

1. Clone the repository

git clone https://github.com/nmnnrb/shelfEx-job-crm-backend
cd shelfEx-job-crm-backend

2. Install dependencies

npm install

3. Start the development server (Vite)

node server.js

4. Open the app

http://localhost:8080

Prerequisites
-------------

- Node.js 18+ (LTS recommended)
- npm (comes with Node) or any compatible package manager
